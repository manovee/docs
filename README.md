# Tamlr Help Center

This repository contains the Mintlify documentation site for Tamlr end users.

## Structure

- `docs.json` configures the Mintlify site, navigation, and branding.
- `index.mdx` is the help center landing page.
- `kb/` contains the end-user knowledge base articles.
- Screenshots should be added only after redaction or when publishing to a private docs repository.

## Local Preview

Install the Mintlify CLI and run the docs site from this repository:

```bash
npx mintlify dev
```

## Article Scope

The KB covers getting started, free visibility checks, dashboard interpretation, prompts, AI tests, source domains, competitors, products, settings, team management, CMS connections, admin workflows, troubleshooting, and glossary terms.
